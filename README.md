# 💿 MySpace-Style Portfolio

This is a nostalgic, glittery, MySpace-inspired personal portfolio site you can customize and host via GitHub Pages.

## 🚀 Getting Started

To get your site live:

1. **Clone or download this repo**
2. Customize your content (see below)
3. Push to a GitHub repository
4. Enable GitHub Pages in your repo settings

---

## 🛠 Customization Guide

### 1. **Profile Picture**
- Replace `assets/profile.jpg` with your own image.

### 2. **Projects**
- Add images to the `assets/` folder.
- Edit project titles, images, and descriptions in `index.html` under the `<section id="portfolio">`.

### 3. **About Me**
- Update the `<section id="bio">` with your own bio text.

### 4. **Contact Info**
- Update links in `<section id="contact">` with your real email, GitHub, and LinkedIn.

### 5. **Music**
- Replace `assets/music.mp3` with your own `.mp3` file.
- Change the filename in the `<audio>` tag inside `index.html`.

### 6. **Style**
- Modify `styles.css` to tweak layout, colors, borders, etc.
- Background glitter, MySpace font (`Orbitron`), and sparkle effect are all included.

### 7. **Sparkle Effect**
- `scripts.js` controls the sparkle overlay.
- You can swap `assets/sparkle.gif` with any repeating transparent glitter gif.

---

## 🌐 Deploy to GitHub Pages

1. Push your folder to a GitHub repo (e.g., `myspace-portfolio`)
2. Go to **Settings** → **Pages**
3. Under **Source**, choose `main` and root `/`
4. Visit `https://yourusername.github.io/myspace-portfolio`
